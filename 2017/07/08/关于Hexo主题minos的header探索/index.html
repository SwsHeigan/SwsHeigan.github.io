关于Hexo主题minos的header探索

将themes里的 _source 里的文件夹放入 site 的 source里面，只是确保 有页面产生。

而我们是没有出现菜单栏。

minos的readme所说的：

To enable custom categories page, tags page and about page, just copy the categories folder, tags folder and about folder under your theme’s _source foler into your site’s source folder. Then edit theme’s _config.yml and add the following lines

最后一句，可能是错的。

本人的解决方法是 经查网上minos 相同使用者 [lor’s blog] 所言，将 所给的：

直接放入 【site】 的 _config。而不是 【themes】的 _config。

从而出现了 菜单栏。

这次问题的发现与解决用了我两天时间。期间换用了其他主题，后来 还是发现 不如 minos+我自己做的logo 配合起来好看。

以后还将对主题做出更多修改，对网上的 hexo 学习 还将继续。
