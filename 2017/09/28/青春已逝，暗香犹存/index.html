青春已逝，暗香犹存

温习曾经的文字，似乎就是在召唤淡化了的悲伤，反衬现时的萧瑟。不怕寂寞，却在不知不觉将寂寞无限量的放大，渐渐再次触然悲伤的边缘。

— —引

一直都渴望能够展翅拥抱属于自己的一片蓝天，安静的飞翔，凌空于混杂的人世，守着自己，守住曾经的渴望与信仰。

　　又或安逸在时光流年中，半睡半醒，迷离在若有似无的旧梦中。

万物皆空，无所谓忧伤，也不知如何对比欢乐的模样。只是情绪时常也会打乱时间的节奏，跌落在某个时间的节点上！

　　停滞在时光的结点上，回眸曾走过的年华。谈不上什么物是人非，只是曾经生机盎然、富有活力的理想与笃定在时间的侵蚀下，凋落在风中，随风而逝。留有半丝无奈，一缕讥笑。

　　很多人都说青春是一场华丽的盛宴，可我并没感到生活因青春而有任何大的起伏，记不起它有轰轰烈烈的曾经，或许有，但已归于平淡。只是漫长突然短暂，还没有为梦想指引一个清晰的方向，出发的号角已经响彻脑海。

　　青春梦想的结局隐含在未知的明天，明天正在不明的世界里酝酿，谁也猜测不了。无论改变成什么模样，也不会看透未知的结局。

也有人哀怜地说青春是一场刻苦铭心的伤，青春就是失去纯真、收获迷茫的过程。

　　我不怕失去，心痛或惋惜也只是承受的过程，纵然狼狈，纵然心痛到无法忍受，结局都还是忍受。淡然地迷茫在红尘中，蹒跚于生活的曲线，走走停停，给快乐留一些享受的时间，也给悲伤腾出些放任的空间……

　　曾经以为自己是红尘最深处的那双不谙世事的眼睛，明晰世间一切感悟，以为自己在孤独中洞察眼前掠过的每个人物，淡淡地看着世间的纷繁嘈杂。
　而其实在流动、荒乱的生活中，我们谁也说不清为何在不停的迁移。陌生、熟悉、相识，然后再见，我们在反复串演彼此生活中过客。不是不知道疲惫，只是我们都已经习惯了告别熟悉，然后适应陌生。

　　不知道究竟有没有轮回这一回事，但我们的确在反复某些事情。一程经历，一场感悟；一次轮回，一种懂得。只如清风，轻抚万物不留痕，吹遍世间难寻迹，潇洒走一回。

　　世事纷繁终无言。如果真的执着，那就是没有结局，执着的意念又是为何？生命中的抵达似乎在一个又一个终点面前颓靡，直到生命的终止，那或许才是真正的抵达。

　　相知轮回，淡写忧伤。人生中的确有无数的看客与过客，但我也一直相信，有些情感是注入在生命里的，有些人是驻扎在你的生活中的。

人生在世，悲欢离合、喜怒哀乐都是生活的形态，我们逃不过任何一种情绪。

　　所有的忧伤欢乐伴随着生活化为平淡，融入生命的血液，那是不可或缺的独享时刻，自由的放任自己，不管心的沉沦或是心的狂喜，只要畅快淋漓，一个人的畅快淋漓。

　　青春正在凋落，悲伤早已衰老。衰老后的忧伤是淡然，冷却蔓延的热情，淡然的无知无觉，曾经的我们也正消逝在无声的时光中。

　　这世界上不存在哪一个角落里的时光是永不荒凉的，时光也不会永久地萧条在某一隅。

记忆在岁月里回荡，暗香犹在。成长的责任在现实里猖狂，躁动不安。
